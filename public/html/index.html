Timestamp Microservice API

Made for FreeCodeCamp's Timestamp Microservice challenge.

Example usage and result:

https://hj-timestamp.herokuapp.com/21%20Jan%201981
{"unix":348883200,"natural":"January 21, 1981"}

https://hj-timestamp.herokuapp.com/Dec%203%202011
{"unix":1322870400,"natural":"December 3, 2011"}

https://hj-timestamp.herokuapp.com/Jun1997
{"unix":865123200,"natural":"June 1, 1997"}

https://hj-timestamp.herokuapp.com/8293171671
{"unix":8293171671,"natural":"October 19, 2232"}
